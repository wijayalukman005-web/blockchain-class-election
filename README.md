# Blockchain Class Election

A decentralized e-voting smart contract built with Solidity on the Ethereum blockchain.

## Overview

This project demonstrates a simple blockchain-based election system for selecting a class leader. The voting process is transparent, secure, and prevents double voting by allowing each wallet address to vote only once.

## Features

- Add election candidates
- One wallet = one vote
- Automatic vote counting
- View candidate information and vote totals
- Transparent and immutable voting records
- Deployable on Ethereum Sepolia Testnet

## Technologies Used

- Solidity 0.8.20
- Remix IDE
- MetaMask
- Ethereum Sepolia Testnet

## Smart Contract Functions

### addCandidate()
Add a new election candidate.

### vote()
Cast a vote for a candidate.

### getCandidate()
View candidate details and total votes.
